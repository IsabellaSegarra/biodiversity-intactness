## Data access
The data for this study includes Arizona geographic information accessed through the [U.S. Census Bureau](https://catalog.data.gov/dataset/tiger-line-shapefile-2023-state-arizona-az-county-subdivision). Download the zip folder titled tl_2023_04_cousub.zip. The data for the Biodiversity Intactness Index (BII) is from the [Microsoft Planetary Computer](https://planetarycomputer.microsoft.com/dataset/io-biodiversity). This is accessed directly through the catalog search `io-biodiversity` and requires no external data upload.
